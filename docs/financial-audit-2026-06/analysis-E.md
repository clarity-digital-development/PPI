# PPI COMPETITIVE LANDSCAPE & GAP ANALYSIS
**Pulled 2026-06-08. National + local pricing benchmarks vs PPI's current $63.95 install AOV.**

## TL;DR

1. **PPI has no direct local competitor in central KY** doing the full install-and-pickup-as-a-service model. Lexington/Louisville's sign companies (FASTSIGNS, SpeedPro, Image360, Sherrod, Signarama) are **print shops** that install on request — not subscription-style operators. PPI's $63.95 install AOV is **20-50% below national install-specialist pricing.**
2. **National peers price install at $55-$85.** SignBoss (Houston) $75, KWPosts (Ontario) $55, True Sign (Toronto) ~$85, **InstallNow.io $79 — and they operate in Cincinnati**, which is one of PPI's 5 service centers. They're the only direct overlap and they're priced **24% higher than PPI**.
3. **Add-on fees are where competitors actually make money.** Oakley charges $22 for rider swaps, $24 for solar lights, $19 for brochure box rental, $75 for lost posts, $25 OOA surcharge. PPI under-prices nearly every add-on (riders $4.88 avg, solar $6.89, lockbox $5.65, brochure box $9).
4. **PPI's "full-service bundle" is the moat**, but it's being given away as commodity install. The bundle (install + storage + pickup + inventory + broker portal + proposal flow) is more sophisticated than any of the comparables. SignBoss is the closest analog and they charge $120/yr subscription on top of per-action fees.
5. **Software integration gap exists.** kvCORE, BoomTown, CINC, BoldTrail have no integrated sign ordering. None. This is a wide-open category to be first in.

## 4. THE UNBUNDLING QUESTION

PPI bundles: **manufacture (riders) + storage + install + rental + pickup + broker portal + proposal/invoicing**. Competitors unbundle this:
- **Manufacturers** (Dee/Lowen) sell signs, no install
- **Install specialists** (InstallNow/SignBoss/Oakley) install but customers bring their own designs
- **Open-house specialists** (Place My Signs at $69.99-$104.99 per event) handle only weekend pop-ups

**Verdict: the bundle IS the moat.** No competitor in central KY offers it end-to-end. PPI's mistake isn't bundling — it's **pricing the bundle like a commodity install** ($63.95) instead of like the full-service hand-off it actually is ($79 InstallNow / $75 SignBoss baseline).

## 8. POSITIONING RECOMMENDATIONS

1. **Reprice the base install $63.95 → $79.** Match InstallNow's Cincinnati rate. The data shows AOV already moved $54 → $76 in June with no resistance. Pricing power exists.
2. **Productize add-ons as SKUs at competitive rates.** Lockbox $20, rider swap $22, solar $20/pair. Brokers (team_admin) keep their exemption — solo agents pay. This alone moves AOV 15-25%.
3. **Compress the 6-month free rental to 3 months, then $12/mo.** Median install life is 35 days, so this rarely fires — but when it does, charge a competitive rate, not the punitive $18 + $6/mo schedule that competitors price flat-monthly.
4. **Launch open-house weekend service ($75 for Sat-Sun, 10 directionals).** Place My Signs proves nationwide demand. PPI already has crew + inventory. Pure margin add.
5. **Charge for every service-request trip.** 40 of 45 service requests in 90 days were unbilled — $40 × 40 = $1,600 left on the table. Auto-invoice on dispatch, refund on cancel, not the other way around.
6. **Sell a Broker Portal subscription ($249/yr).** SignBoss proves the $120 subscription tier works. PPI's broker tooling is more sophisticated. Semonin's dormant team_admin proves the seat-of-power exists; monetize it.
7. **Build a kvCORE/BoomTown/BoldTrail integration.** First-mover in CRM-integrated sign ordering. Becomes a moat against any new entrant who tries to copy the install model in KY.
8. **Don't chase the out-of-area surcharge.** No competitor has captured demand outside the 5-center band either — there's no there there.
